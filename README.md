# Mindspace

**Install all dependencies**

	$ npm install
	$ bower install

**Build and watch**

	$ gulp watch

**Deploy locally**

	$ gulp build_guide
	$ gulp serve
	
**Serve cms**

	$ npm install -g wh
	$ wh init unitid-mindspace
	$ npm install
	$ wh serve
	
Once all changes done, you can use, to push the changes:

	$wh deploy
